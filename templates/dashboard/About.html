Food Pantry Project


  • We created a new Django project and the required apps for it, i.e dashboard, waste_management, provider, inventory and impact_measurement. We started with the login and register page. After that the dashboard.html and Navigation_Bar.html to include the navigation bar, and also shared_links.html to include the various link of bootstrap.
  • After proper function of Login and logout, including Registration, It's time to work on the provider app. We created the edit_task page and add_task for it and also added the various functions to implement the tasks app. I also added the Hide complete Tasks checkbox as per the requirement and gave the final touches with the CSS.
  • After Making a commmon CSS file for the project and one more CSS for the nav bar, Its easiler to impletmet the CSS in whole project, It also gives the uniformity to pages and the website looks very aesthetic. After completing the dashboard app, we worked the provider app, created tables and implemeted the edit and delete for provider as required for the project.
  • After completing the provider app, we worked on the inventory, so that once a provider is added, we can add items in the inventory by choosing the provider at first. The same delete and edit functions are used in inventory too.
  • Third app, we worked on is the Check out. As items in inventory needs to be checked out by a student, it will transfer the item name and the quantity add in cart will be subtracted from the inventory.
  • With all the functions working, We worked on the app like impact_measurement and waste_management. There apps are for the statistics of the whole food pantry. In the impact_measurement, statistics like average number of items is displayed by chartist.js and the whole table of students and their number of items checked out is displayed.
  • In the waste_management, We made two tables, one of which shows the items which are going to expire in two days and the another one show the expired items, that needs to be dumped in the garbage.
